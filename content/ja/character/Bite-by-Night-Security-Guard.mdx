## Bite by Night 警備員のアビリティを使いこなす

このクラスの強みは、2つのユニークなアクティブアビリティ「テイザー」と「カメラシステム」にあります。他のクラスが修理速度や隠密に焦点を当てる一方で、**Bite by Nightの警備員**は脅威や環境と直接対話するように設計されています。

### 1. テイザー（スタンメカニック）
テイザーは主要な防御ツールです。アニマトロニクスをスタンさせるか、大幅に移動速度を低下させる発射体を放ちます。このツールの最も重要な側面は「チャージ」メカニックです。発射ボタンを長く押し続けるほど、電気放電が強力になります。

| チャージレベル | アニマトロニクスへの効果 | 戦術的用途 |
| :--- | :--- | :--- |
| **クイックファイア** | わずかな鈍化 | 近距離でのチェイス中の緊急自己防衛。 |
| **中チャージ** | 短いスタン | キラーがバリケードを破壊するのを阻止する。 |
| **フルチャージ** | 大幅なスタン / 停止 | 捕まりそうなチームメイトを救出する。 |

> **💡 ヒント:** 極端に遠い距離でテイザーを無駄撃ちしないでください。射程距離はそれなりにありますが、発射体には到達時間があり、外すとクールダウンが終わるまで無防備な状態になってしまいます。

### 2. カメラ監視（情報収集）
情報はしばしば生の力よりも価値があります。「カメラ」アビリティにアクセスすることで、マップ各所に配置された固定カメラを通路を確認できます。これにより、自分を物理的な危険にさらすことなくキラーの動きを追跡できます。

*   **アニマトロニクスの可視化:** カメラでキラーを見ているときにクリックすると、チーム全員に対してキラーの位置がハイライトされます。
*   **発電機の特定:** カメラを使用して未完了の発電機を見つけ、チームがより効率的に修理を調整できるように支援できます。
*   **安全確認:** メインホールのような通行量の多いエリアに入る前に、近くのカメラフィードをチェックして安全を確認しましょう。

## 戦術的なゲームプレイ戦略

**Bite by Nightの警備員**としてプレイするには、考え方の転換が必要です。あなたは単なる発電機の技術者ではなく、チームの目であり耳です。効果を最大化するために、マッチ中に以下の戦略を実行してください。

### 足止めの技術
キラーがチームメイトを追いかけているとき、あなたの仕事は介入することです。テイザーを使ってチェイスを中断させましょう。キラーがバリケードやドアを壊すのに忙しいなら、その時間を使ってテイザーをフルチャージしてください。アニメーションが終わる瞬間にフルチャージを放てば、キラーを苛立たせ、チームメイトが新しいループ場所や隠れ場所に到達するのに十分な時間を与えられます。

### スタミナ管理
警備員には無限の持久力はありません。テイザーを当てるために「危険地帯」にいることが多いため、スタミナバーを注意深く監視する必要があります。ショットを狙っている間にスタミナを完全に使い果たさないでください。もし外してしまった場合、近くの窓やパレットまでスプリントするためのエネルギーを残しておく必要があります。

### 戦略的なカメラの使用
カメラに時間をかけすぎないでください。情報は不可欠ですが、隅っこでタブレットを見ているサバイバーは、発電機を修理していないサバイバーでもあります。カメラは短時間で使用しましょう：
1. キラーの位置を確認する。
2. チームのためにタグを付ける。
3. フィードを終了して目的の作業に戻る。

> **⚠️ 警告:** カメラを使用している間は周囲の状況に注意してください。フィードを見ている間は目の前の環境が完全に死角になるため、隠密性の高いアニマトロニクスにとって格好の標的になります。

## チームのシナジーと構成

**Bite by Nightの警備員**は、目的達成速度に焦点を当てたクラスと組んだときに最高のパフォーマンスを発揮します。キラーの注意を逸らし、位置を可視化し続けることで、「エンジニア」や「修理スペシャリスト」が不意打ちを恐れることなく発電機を完成させることができます。

| シナジーパートナー | 効果的な理由 |
| :--- | :--- |
| **エンジニア** | 彼らが高速修理を終える間、あなたがキラーを足止めします。 |
| **メディック** | ダウンした仲間を回復させている間、メディックを保護します。 |
| **スカウト** | あなたのカメラと組み合わせることで、キラーはほぼ常にマップ上に表示されます。 |

連携の取れたグループでプレイする場合は、保護のために1人を**Bite by Nightの警備員**の近くに配置しましょう。テイザーがクールダウン中のときにパレットを倒したり注意を引いたりしてくれる2人目のプレイヤーがいれば、キラーが突破するのが非常に困難な「ループ」を作り出すことができます。

## FAQ

**Q: 警備員は900スクラップの価値がありますか？**

A: はい、**Bite by Nightの警備員**は、キラーをスタンさせる能力とマップ全体の情報を共有できる能力の両方を備えているため、ゲーム内で最高のサポートクラスの1つと広く見なされています。戦術的なプレイを楽しむプレイヤーにとって、優先的に解放すべきクラスです。

**Q: テイザーの弾薬は切れますか？**

A: テイザーは限られた弾薬ではなく、クールダウンシステムで動作します。発射後は、デバイスが再チャージされるまで待つ必要があります。このクールダウンを管理することが生存に不可欠です。

**Q: 「可視化」メカニックは壁越しでも機能しますか？**

A: カメラシステムを通じてアニマトロニクスをクリックすると、そのシルエットやマーカーが一定時間、壁越しでもチームメイトに見えるようになります。これにより、回避や移動がよりスムーズになります。

**Q: キラーはカメラを無効化できますか？**

A: 一部のキラーは、カメラフィードを一時的にジャミングしたり無効化したりする特定の能力を持っています。画面にノイズ（砂嵐）が見える場合は、キラーが近くにいるか、妨害能力を使用している兆候ですので、すぐに場所を移動する必要があります。
