## エレベーターとロビーの秘密

旅はマッチにロードした瞬間から始まります。プレイヤーはメインロビーへと向かう下降するエレベーターの中でスポーンします。シリーズの長年のファンなら、これが『FNAF: Sister Location』のエレベーターを忠実に再現したものであることにすぐに気づくでしょう。象徴的な回転ファン、独特の金属製のレイアウト、そしてオリジナル作品のトーンを決定づけた閉所恐怖症のような雰囲気が特徴です。

プレイヤーは隠しフロアをアンロックするための物理的な **Bite by Night Elevator Code** を探すことが多いですが、エレベーター自体はロビーのインタラクティブな機能へのゲートウェイとして機能します。到着後は、いくつかのプレイ可能なアーケードマシンで時間を潰すことができます。

| 機能 | インタラクション | 秘密の詳細 |
| :--- | :--- | :--- |
| **アーケードマシン** | **E**キーでプレイ | すべてのマシンで現在 **Snake（ヘビゲーム）** が動作しています。 |
| **Sister Location風リフト** | 自動降下 | Afton Roboticsのデザインを高品質に再現しています。 |
| **ロビー・ハングアウト** | ソーシャルエリア | マッチ開始前にプレイヤー同士で連携を取るための場所です。 |

> **💡 ヒント:** ロビーにいる間に移動キーのテストをしておきましょう。アーケードゲームは単純ですが、フルロビーを待つ間のウォーミングアップに最適です。

## ピザ屋マップ：警備室とバックステージの伝承

ピザ屋マップでマッチが始まったら、生存者は警備室に向かうべきです。このエリアはFNAFの第1作目への直接的なオマージュであり、機能するドアとライトが完備されています。施設内の他の場所にある標準的なドアとは異なり、これらのセキュリティドアは破壊不可能であるため、キラーは別のルートを探す必要があります。

しかし、生存には代償が伴います。エネルギーバーを慎重に管理しなければなりません。ドアを閉めたりライトを点けたりすると、電力を消費します。2026年版では、エネルギーが切れると完全に無防備な状態になってしまいます。

### 隠されたアニマトロニクス収容セル
メインステージの裏側には、2つの収容セルがある制限エリアがあります。これらのセルは、フルロースターが開発されるずっと前の、初期のキャストのために設計されたようです。

*   **セル1（左）:** **Freddy Fazbear（フレディ・ファズベアー）** 用のラベル。
*   **セル2（右）:** **Spring Bonnie（スプリング・ボニー）** 用のラベル。

これらのセルには重い鎖とフックがあり、施設ができたばかりの頃、アニマトロニクスを収容するためにより工業的で、おそらく暴力的な方法がとられていたことを示唆しています。このエリアは、伝承における「チカとボニー以前」の時代を垣間見せてくれます。

## 森林とキャンプマップの謎

森林マップは、ピザ屋の工業的な雰囲気とは対照的です。環境ストーリーテリングに満ちた広大な屋外エリアです。最も議論されている場所の一つは、孤立したトレーラーです。

### 謎のトレーラーと隠された家
トレーラーには弾痕だらけのドアがあり、「ノックしてサプライズを（Knock for a surprise）」という看板が掲げられています。現在、プレイヤーはこのドアに干渉することはできませんが、将来のアップデートのティーザー、あるいは『ブレイキング・バッド』のような人気メディアへのリファレンスであると広く推測されています。

近くのフェンスで囲まれたエリアには、プレイヤーが立ち入ることのできない家があります。しかし、高倍率のズームツールやスクリーンショットを使用することで、プレイヤーは玄関ドアに書かれた不気味なメッセージを発見しました：**「It's Me（ボクだよ）」**。この象徴的なフレーズの下には、解読不能な茶色のテキストが追加されており、将来のARG（代替現実ゲーム）要素の不可解な手がかりである可能性があります。

### キャンプ場の絵
キャンプ場では、子供が描いたような絵に囲まれたテディベアを見つけることができます。これらのスケッチは、2026年の物語を理解する上で非常に重要です：
1.  **少女と4人の子供:** 泣いている4人の子供と一緒にいる少女。
2.  **パペット:** 同じ子供とマリオネット（Marionette）の描写。
3.  **幸せな家族:** 父、母、子供の絵。施設が土地を乗っ取る前に、ここに家族が住んでいたことを示唆しています。

## キラーのメカニクス：ミミックとスプリングトラップ

キラーとしてプレイすると、ユニークなゲームプレイが楽しめます。最も革新的な機能の一つは、リアルタイム・ボイスチェンジャーです。キラーとしてマイクで話すと、ゲームがロボットのような歪んだフィルターを声に適用し、キャラクターモデルの口が話に合わせて動きます。

### ミミックの3つの脅威モード
ミミック（The Mimic）は、**Q**キーを押すことで3つの異なる動作モードを切り替えられる多才なキラーです。これらのモードを理解することは、マップナビゲーションのために **Bite by Night Elevator Code** を知ることと同じくらい重要です。

*   **ストレングス（パワー）モード:** デフォルトの状態。バランスの取れたステータスを提供し、強力なランジアタックが可能です。
*   **スピードモード:** 移動速度が大幅に上昇しますが、ダメージが75%減少します。離れた生存者との距離を詰めるのに最適です。
*   **ステルスモード:** 地下に潜ることができます。潜伏中は監視カメラに映りません。生存者の真下に飛び出すと、相手をノックバックさせ、中程度のダメージを与えます。

### スプリングトラップの隠されたディテール
スプリングトラップ（Springtrap）は、ウィリアム・アフトンの運命の凄惨な現実を明らかにする非常に詳細なモデルが特徴です。特定のキルアニメーション中、彼のマスクが引き剥がされ、恐ろしいほどリアルな頭蓋骨が露出します。筋肉構造や腹筋の定義さえ見える比較的「きれいな」胴体とは対照的に、顔は腱と骨が腐敗した無残な姿になっており、原作のホラーのルーツに忠実です。

マップの各所に散らばっているアーケードマシンは、ただの飾りではありません。開発者のユーモアのセンスや潜在的な伝承の手がかりを暗示する奇妙なメッセージが含まれています。

| マシン名 | メッセージ/謎 | 潜在的な意味 |

## FAQ

**Q: Bite by Night Elevator Code はどこで見つけられますか？**

A: 現在の2026年バージョンのゲームでは、エレベーターを操作するために必要な数字キーパッドのコードは存在しません。「コード」という言葉は、多くの場合、プレイヤーがゲームの隠された伝承を「解読」するために使用する、「It's Me」のドアや倉庫のステータスボードにある秘密のメッセージを指します。

**Q: ミミックのステルスモードを生き延びるにはどうすればいいですか？**

A: 動き続けてください。ミミックがあなたの真下に正確に飛び出すには、数秒間静止している必要があります。低い機械的な研磨音が聞こえたら、潜伏中のミミックが近くにいる可能性が高いです。

**Q: キラーは警備室のドアを壊せますか？**

A: いいえ。ピザ屋マップでは、FNAF 1スタイルのセキュリティドアは破壊不可能です。しかし、これらは限られた電力供給に依存しています。エネルギーが0%になると、ドアは故障し、マッチの残り時間はずっと開いたままになります。

**Q: スプリング・ボニーの「脱走（Breached）」ステータスはどういう意味ですか？**

A: 倉庫の隠し部屋で、スプリング・ボニーは「Breached」と記載されています。これはアニマトロニクスが収容ユニットから脱走し、施設内を徘徊していることを意味しており、プレイ可能なキラーとして存在している理由を説明しています。
