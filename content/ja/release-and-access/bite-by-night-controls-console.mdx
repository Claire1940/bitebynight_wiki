## サバイバーの操作と移動メカニクス

Bite by Nightでサバイバーとしてプレイすることは、隠密行動、リソース管理、そして素早い反射神経のテストです。主な目標は、キラーを避けながら目的を達成することであり、そのため移動操作は最も貴重な資産となります。PCでは伝統的なWASD形式を採用していますが、スタミナや多目的キーの導入により、操作に深みが加わっています。

| アクション | PC キー割り当て | コンソール対応ボタン (Xbox/PS) |
| :--- | :--- | :--- |
| **移動** | WASD | 左アナログスティック |
| **ダッシュ** | 左 Shift | L3 (左スティック押し込み) |
| **しゃがむ** | C | B / 〇ボタン |
| **インタラクト** | E | X / ▢ボタン |
| **懐中電灯 / バリケード** | F | Y / △ボタン |
| **スキル 1** | Q | L1 / LB |
| **スキル 2** | E | R1 / RB |

> **💡 ヒント:** ダッシュはスタミナを急速に消費します。ダッシュキーを「連打」するのは避け、追跡されている時や開けた場所を素早く横切る必要がある時のために、瞬発的なスピードを温存しましょう。

サバイバーにとって「F」キーは特にユニークです。これは懐中電灯の切り替えとドアのバリケード設置という2つの目的を果たします。懐中電灯は暗い角を探索するのに不可欠ですが、キラーにとっては目印となってしまいます。居場所を察知されないよう、控え目に使用しましょう。出入り口の近くに立っている時に同じキーを押すと、バリケードを設置して追跡者との間に一時的な盾を作ることができます。

## キラーの操作とアビリティ管理

キラーのゲームプレイサイクルは、追跡と排除に重点を置いた全く異なるものです。基本的な移動は同じですが、キラーの「E」および「Q」キーは環境操作や生存ツールではなく、殺傷能力のあるアビリティ専用となっています。キラー用の**Bite by Night操作コンソール**レイアウトは、より効率的に設計されており、プレイヤーが狩りに集中できるようになっています。

## インタラクトキーの詳細解説

世界と相互作用（インタラクト）することは、サバイバーがゲームの目的を進行させるための手段です。発電機を修理する、クレートを探索する、あるいはダウンした仲間を助ける場合でも、「E」キー（コンソールではX/▢ボタン）が主要なツールとなります。

### 環境への干渉
環境内にはインタラクト可能なオブジェクトが溢れています。効率的なプレイには、これらの干渉の優先順位を理解することが不可欠です。例えば、発電機とロッカーの両方の近くに立っている場合、ゲームは照準に最も近いオブジェクトを優先します。

| オブジェクトの種類 | キー | 結果 |
| :--- | :--- | :--- |
| **目的アイテム** | **E** | マップのメイン目標を進行させる |
| **ロッカー** | **E** | 隠れることができる（「しゃがむ」と併用） |
| **ドア** | **E** | 通路を開閉する |
| **バリケード** | **F** | 物理的な障害物を設置する |

> **⚠️ 警告:** 目標にインタラクトすると、しばしば大きな音や視覚的な合図がキラーに送られます。長時間のインタラクトを開始する前には、必ず周囲を確認してください。

## 高度なコンソール操作と最適化

**Bite by Nightの操作コンソール**設定を利用しているプレイヤーにとって、感度管理は生死を分けるポイントです。コントローラーはマウスではなくアナログスティックを使用するため、旋回速度は設定によって制限されます。

1.  **カメラ感度の調整:** ゲーム内設定メニューに移動し、「カメラ感度（Camera Sensitivity）」スライダーを見つけます。サバイバーは、チェイス中に素早く背後を確認するために感度を高めに設定すると有利です。
2.  **デッドゾーンの調整:** キャラクターが勝手に動いたり視点が動いたりする場合は、スティックのデッドゾーンを上げて「スティックドリフト」が隠密行動を台無しにするのを防ぎましょう。
3.  **ボタンマッピング:** Robloxゲームのレイアウトは固定されていることが多いですが、デフォルトのインタラクトボタン（X）が不自然に感じる場合は、コンソールのシステム設定でカスタムボタン割り当てが可能か確認してください。

PCからコンソールへの移行は、スキルの割り当て方法の違いにより戸惑うかもしれません。PCではQとEが移動キーのすぐ隣にあります。コントローラーでは通常これらがバンパー（LB/RBまたはL1/R1）に割り当てられており、常に親指をスティックに置いたまま操作できます。これは激しいチェイスにおいて、コンソールプレイヤーのメカニカルな利点となります。

## スタミナと隠密戦術

プラットフォームに関わらず、スタミナバーは最も制限の多いリソースです。Bite by Nightでは、走るよりも隠密行動の方が効果的な場合が多いです。しゃがむ（PCではC、コンソールではB/〇ボタン）ことで姿勢を低くするだけでなく、足音を消すこともできます。

| 移動状態 | 騒音レベル | スタミナ消費 |
| :--- | :--- | :--- |
| **しゃがむ** | **無音** | なし |
| **歩く** | **低い** | なし |
| **ダッシュ** | **高い** | 高い |

**Bite by Nightの操作コンソール**を使用してこれらの状態を素早く切り替えるのは、時間をかけて習得するスキルです。サバイバーによくある戦術は、キラーの視線を切るまで走り、すぐにしゃがんでオブジェクトの陰に隠れることです。この「ストップ・アンド・ゴー」の動きは、キラーの聴覚情報を混乱させ、次の遭遇のためにスタミナを温存することができます。

## よくある操作トラブルの解決

現在の2026年ビルドでは、遅延の激しいマッチにおいて「入力ラグ」やキーが反応しないという報告が時折あります。「E」キーによるインタラクトがリセットされてしまう場合は、タップするのではなくボタンをしっかりと押し続けているか確認してください。ほとんどの目的は「長押し」で完了させる必要があるためです。

コンソールでプレイ中にUIプロンプトがPCのキー（「Press E to Interact」など）を表示していても、驚かないでください。これはプラットフォームをまたぐRobloxタイトルではよくある現象です。「E」の表示はほとんどの場合、コントローラーの「西」ボタン（XboxのX、PlayStationの▢）に対応しています。同様に「F」の表示は「北」ボタン（XboxのY、PlayStationの△）に対応しています。

## FAQ

**Q: Bite by Nightの操作コンソールのレイアウトは変更できますか？**

A: 現在、ゲーム内での完全なボタン割り当て変更機能はコンソール向けに提供されていません。ただし、別のレイアウトを好む場合は、コンソールのシステムレベルのアクセシビリティ設定（XboxアクセサリアプリやPlayStationの設定）を使用して、ボタン機能を入れ替えることができます。

**Q: 懐中電灯が自動的に消えるのはなぜですか？**

A: Bite by Nightの懐中電灯にはバッテリー寿命があります。長時間点灯したままにすると、電池が切れます。また、特定のキラーのアビリティを受けると、光源を落としたり解除したりすることがあります。

**Q: コンソールでキャラクターのスペシャルアビリティを使うにはどうすればいいですか？**

A: スペシャルスキルやクラスアビリティは、通常ショルダーボタン（LB/RBまたはL1/R1）に割り当てられています。PCではQおよびEキーです。発動を試みる前に、アビリティのクールダウンが終わっていることを確認してください。

**Q: コントローラーでダッシュしながら後ろを向く方法はありますか？**

A: はい、左アナログスティックを前に倒したまま、右アナログスティックを使用してカメラを回転させることができます。これは進行方向を変えずに、チェイス中のキラーとの距離を把握するために不可欠なスキルです。
