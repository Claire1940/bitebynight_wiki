## Entendendo o Valor dos Scraps

Em Bite by Night, Scraps são muito mais do que apenas um número digital; eles são o principal motor de progressão. Ao contrário de alguns jogos onde os cosméticos são o único uso para a moeda, este jogo integra os Scraps no ciclo principal de jogabilidade.

### Onde Gastar Seus Scraps

1.  **Desbloquear Novos Sobreviventes**: Diferentes sobreviventes vêm com habilidades passivas únicas. Alguns podem correr mais rápido quando feridos, enquanto outros podem reparar objetivos de forma mais silenciosa.
2.  **Adquirir Assassinos**: Para jogar como o antagonista, muitas vezes você precisa comprar assassinos específicos com Scraps. Cada assassino tem um conjunto distinto de poderes que requer diferentes estratégias para dominar.
3.  **Upgrades de Personagem**: Use seus Scraps para subir o nível de seus personagens existentes, aumentando a potência de suas vantagens (perks).
4.  **Skins e Emotes**: Embora não ofereçam vantagem na jogabilidade, as skins são uma ótima maneira de mostrar seu status de veterano no lobby.

## Resolução de Problemas Comuns

Mesmo com um guia claro, você pode encontrar obstáculos ao tentar resgatar suas recompensas. Aqui estão as soluções mais comuns para problemas relacionados aos **códigos de bbn**.

*   **"Code Already Redeemed" (Código já resgatado)**: Isso significa que sua conta já utilizou este código específico. A maioria dos códigos é de uso único por conta do Roblox.
*   **"Expired Code" (Código expirado)**: Os códigos têm um tempo de vida. Se você está tentando usar um código de 2025, ele quase certamente está inativo.
*   **Lag do Servidor**: Se você resgatar um código e seus Scraps não aparecerem imediatamente, tente entrar em um novo servidor. Às vezes, os dados levam um momento para sincronizar com o banco de dados global.
*   **Erros de Digitação**: Evite a digitação manual, se possível. Copie e cole os códigos de uma fonte confiável como este artigo para evitar espaços acidentais ou troca de caracteres.
