## Como Resgatar Códigos no Bite by Night

Resgatar recompensas neste jogo é simples, mas o menu pode estar um pouco escondido para jogadores acostumados com botões de "Código" mais proeminentes no HUD principal. Siga estas etapas para reivindicar seus Scraps:

1. **Inicie o Bite by Night** na plataforma Roblox.
2. Localize o ícone de **Configurações** (representado por uma engrenagem) no menu principal ou no painel lateral.
3. Clique no ícone para abrir a janela de configuração.
4. Encontre a caixa de texto rotulada como **"Enter Code"** (Inserir Código) ou similar.
5. Digite ou cole seu **código de bite by night** no campo.
6. Pressione a tecla **Enter** ou clique no botão **Redeem** (Resgatar) para reivindicar seus 1.000 Scraps.

## Solução de Problemas Comuns com Códigos

Se você estiver tendo problemas para fazer um código funcionar, considere os seguintes erros comuns:

1.  **Status Expirado:** Os códigos em Bite by Night geralmente estão vinculados a versões específicas do jogo. Se o jogo foi atualizado desde que o código foi lançado, ele pode não ser mais válido.
2.  **Erros de Formatação:** Certifique-se de que não haja espaços extras no final do código se você estiver copiando e colando.
3.  **Uso Único:** Cada **código de bite by night** só pode ser resgatado uma vez por conta. Se você já reivindicou os 1.000 Scraps de "Sorry guys", tentar inseri-lo novamente resultará em erro.
