2.  **Hidden Spaces:** If you copy and paste a code, ensure you didn't accidentally include a space at the beginning or end of the string.
3.  **One-Time Use:** Every code can only be redeemed once per account. If you have already claimed the 1,000 Scraps, trying the code again will result in an error.
4.  **Server Version:** Occasionally, brand new codes only work in "New Servers." If a code isn't working, try joining a different lobby.
